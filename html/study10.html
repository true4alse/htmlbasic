자바스크립트

홈페이지의 기능을 만들어줄 때 사용하는 언어

본격적인 프로그래밍 언어

브라우저를 제어하는 언어

데이터형태 / 숫자 문자 참거짓 배열 객체 함수

자바스크립트도 기본적으로는 위에서 아래로 왼쪽에서 오른쪽으로 해석한다.

자바스크립트에서는 ;을 사용하여 명령이 끝났다 라는 것을 명시적으로 표현한다.

코드에서 줄바꿈을 하면 암묵적으로 ;을 사용한 것처럼 인식된다.

숫자데이터형태

사칙연산이 가능한 데이터(+,-,*,/)연산자를 통해서 계산할 수 있다

%연산자는 나머지연산자 (나머지를 계산한다)

문자데이터 형태

문자데이터는 앞뒤로 쌍따옴표 혹은 홑따옴표로 마킹해서 표현한다.

문자데이터 + 문자데이터 => 서로 이어진다.

문자데이터와 숫자데이터를 더하면 이어지면서 문자데이터가 된다.

typeof()도구

typeof()도구는 괄호안의 데이터가 문자인지 숫자인지 검사를 한 뒤 검사결과를 그 자리(typeof도구가 쓰여진자리)에 남기는 도구(남길 때는 문자데이터형태로 남김)
